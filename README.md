# $M Usage 
# ( ClauText includes editor?( https://github.com/vztpv/Console-Data-File-Editor ) )
  1. Up : w or W
  2. Down : s or S
  
  3. Enter : Enter key
  
  4. Back : b
  5. Quit : q
  
  6. Editor mode : e 
      ( add - a, change - c, remove - r, save - s )
  7. Text mode : t ( only at main screen, first screen )
